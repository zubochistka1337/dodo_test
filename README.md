1. uv sync && uv run ./src/main.py --video "./видео 1.mp4"
2. "видео 1.mp4", выбранный столик можно увидеть на output.mp4
3. логика детекции событий: если нижняя часть bbox тела человека(1/3) имеет пороговое пересечение с bbox стола, то считаем что столик занят. Аналогично обратно, если никаких пересечений нет, считаем, что столик свободный. Такой подход выбран из-за 2D ракурса камеры и чтобы условно "голова" человека на переднем плане не мешала столикам на удалении. Поверх этой логики накидан буффер кадров смягчающий ложные детекции, чтоб не было постоянных смен состояний, который можно настроить(по умолчанию 20 кадров). Поверх добавлена логика, что если пересечение человека со столом длится в течении 15 секунд(по умолчанию) + смягчающий буфер кадров, то считаем это сменой состояния. Алгоритм смягчения не идеальный, рядом приложен файл с алгоритмом от нейронки, где добавлена более мягкая логика буферных кадров для смягчения ложных детекций, добавлять к себе уже не стал. Модель базовая yolo11s, тестовое железо: r7 3700x / rtx 5080
